Hero PNG Images
===============
Place hero PNG files here. Each file must match the character ID from characters.json.

Required files:
  aria.png
  kael.png
  sera.png
  torin.png

Recommended size: 128x128 px (or any size — CSS scales it to 128x128)

If a PNG is missing, the game automatically falls back to the built-in
pixel art canvas renderer for that character.
