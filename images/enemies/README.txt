Enemy PNG Images
================
Place enemy PNG files here. Each file must match the enemy ID from enemies.json.

Required files:
  slime.png
  skeleton.png
  golem.png
  dragon.png

Recommended size: 72x84 px (or any size — CSS scales it to 72x84)

If a PNG is missing, the game automatically falls back to the built-in
pixel art canvas renderer for that enemy.
